# 20大答题系统
庆祝20召开，答题练习系统。
- 练习
- 学习
- 考试
## 项目架构
Tomacat+Servlet+Thymeleaf+Mybatis
没有使用任何前端框架，纯手打。
后端使用最原始的Servlet实现。
作为学完Javaweb+JDBC之后的练习项目。
